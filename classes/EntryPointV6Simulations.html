Class EntryPointV6Simulations

Extension of EntryPointV6 that provides gas estimation simulations for user operations. Implements binary search algorithms for estimating verification and call gas limits.

Example

const simulator = new EntryPointV6Simulations(rpcClient);
const gasEstimate = await simulator.estimateVerificationGasLimit({
  userOperation: userOp,
  stateOverrides: overrides
});
